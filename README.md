# Building

## Dependencies

* Node 6+
* Redis 3.2+

## Installing (development)

* Install npm dependencies

```
npm install
```

* Install nodemon

```
npm install nodemon -g
```

* Run the gulp build chain

```
npm run dev
```

* Run the app!

```
nodemon index.js
```
